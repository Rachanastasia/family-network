# Family Network

This project is a work in progress.

## Getting Started

### Environment Dependencies

Install Node.js
Install Yarn
Install Watchman
Install React-Native

### Installation

Clone project

#### Front End
Move to project front-end folder
Install package dependencies with:
'''
npm install
npm audit fix
'''

#### Back End
Move to project back-end folder
Install package dependencies with:
'''
npm install
npm audit fix
'''

### Run
'''
expo start
'''
